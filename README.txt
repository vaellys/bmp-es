#############################################项目启动说明############################################

首先搭建es环境
按照maven项目进行导入
修改src/main/resources目录下的es-config.properties文件中的es.url为自己es服务的地址
使用tomcat进行启动
访问http://localhost:8080/bmp-es/es/index.do进行建索引
访问http://localhost:8080/bmp-es/es/home.do进行查询
